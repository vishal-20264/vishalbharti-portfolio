About Me

I'm a creative researcher and developer passionate about transforming everyday interactions into engaging digital experiences. My work explores tangible interactions, immersive media, and innovative user interfaces to bridge the gap between technology and human experience.

With a background in Electronics & Communication Engineering from IIIT-Delhi, I combine art and technology to create projects that inspire, inform, and innovate.

Projects

Simple Assembler and Simulator

Developed an assembler and simulator for a custom instruction set architecture, enabling machine-level execution of assembly code with precise opcode translation and memory management.

Prototype of a Societal Impact App

Designed and prototyped a mobile application addressing real-world societal challenges. This project involved extensive user research, wireframing, and interactive design using Figma.

Tangible Interactions with Curtain

Engineered an interface that transforms traditional curtains into interactive digital surfaces through sensor-based inputs and dynamic UI feedback. Published (DOI: 10.1145/3681756.3697935).

VoxCraft

Developed voxel-based interactive environments enabling real-time 3D design and manipulation. Work in progress.
